#JavaCC中Token的定义和使用

##1. TOKEN概述
1.1 token是对一个文法的单词结构的名称。

具体来说，它在JavaCC中一般共有3中形态。

第一，表示系统关键字，例如"if"，"else"，"while"等等。

第二，表示字面量，例如128（整数字面量），"abc"（字符串字面量）。

第三，表示分隔符，例如";"或者" "（空格）。

这三种形态构成了我们编程语言中最基本的词汇素材。

1.2 如何在JavaCC中定义Token

````

对应第一种
TOKEN: {
    <VOID:"void">
}

对应第二种
TOKEN: {
 <IDENTIFIER:(["a"-"z","A"-"Z"])+>
 }
 
对应第三种
SKIP : {
    [" ","\t","\n","\r"]
}

````
